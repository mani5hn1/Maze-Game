1. Open SQL Server Management Studio
2. Create database called MazeGame
3. Insert SQL file and Create those tables
4. To connect your table to the Visual Studio
	i. Open Solution File.
	ii. Find the files that consist of a database string and change those values to your values. E.g. sstring connectionString = @"Data Source = (LocalDb)\MSSQLLocalDB;Initial catalog=MazeGame; Integrated Security=True";
	iii. In the Scores Screen, edit the gridview and insert your database to it.
	iv. Clean, Rebuild and Run Solution.
